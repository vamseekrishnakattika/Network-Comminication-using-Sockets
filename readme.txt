The code for the project is in two files, one file for Client and one file for Server. 

The client file is SocketClient.java and the server file is SocketThrdServer.java

Compile and execute instrctions for Server:

To compile the server program use the following command.

javac SocketThrdServer.java

To execute the server program use the following command.

java SocketThrdServer 9000

Here 9000 is the port number

Compile and execute instrctions for Client:

To compile the client program use the following command.

javac SocketClient.java

To execute the server program use the following command.

java SocketClient csgrads1 9000

csgrads1 is the server on which server program is running and 9000 is the port number
